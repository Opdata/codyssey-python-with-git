### 프로젝트 개요

터미널에서 동작하는 나만의 퀴즈 게임 구현
기본 문법을 사용하여 입/출력 흐름을 만들고, 클래스(객체지향)로 코드를 역학별로 구조화 한다.
JSON 파일 저장을 통해 프호그램을 종료해도 퀴즈와 점수가 유지되도록 데이터 영속성을 경험한다.
Git을 통한 변경 이력을 관리
기능 단위로 커밋하며, 브랜치를 나누어 작업한 뒤 병합하며 GitHub에 저장소를 공개한다.
